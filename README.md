# student-performance-analysis
Python project to analyze student performance data using pandas. Calculates totals, averages, and identifies top-performing students from CSV data.
# Student Data Analysis Project

This project analyzes student performance data using Python and pandas.

## Features:
- Reads student data from CSV file
- Calculates total marks and averages
- Identifies top-performing student
- Generates insights from data

## Tools Used:
- Python
- Pandas

## Files:
- student_analysis.ipynb
- students.csv
